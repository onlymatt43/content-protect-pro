🔍 Quick Troubleshooting Checklist

Bunny.net Tests:

  • Can login to Bunny dashboard
  • Video library exists and has videos
  • Token authentication is enabled
  • API key has proper permissions

WordPress Tests:

  • Plugin is activated
  • No PHP errors in logs
  • Database tables were created
  • Gift codes can be created

Presto Player Tests:

  • Presto Player Pro is licensed
  • Can create regular Presto players
  • No JavaScript console errors
  • Player loads without protection

Integration Tests:

  • Protection overlay appears
  • Gift code validation works
  • Video loads after valid code
  • Analytics are recording
